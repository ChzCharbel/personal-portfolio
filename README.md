# Personal Portfolio — Charbel Chávez

Minimalist personal portfolio built with Next.js. Showcases selected projects in machine learning, quantum ML, and AI, along with contact information.

## Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS v4 + inline styles
- **Fonts:** Playfair Display (headings) · Lora (body)
- **Icons:** react-icons

## Structure

```
src/app/
├── components/
│   ├── Navbar.jsx
│   ├── Hero.jsx
│   ├── Projects.jsx
│   └── Contact.jsx
├── page.js
├── layout.js
└── globals.css
```

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Deployed via [Vercel](https://vercel.com). Push to main to trigger a deployment.
